# User Guide

This guide is for tribe members who want to join and participate in a tribe on VITA Frontier.

## Logging In

VITA Frontier uses **Discord** for authentication. No passwords to remember!

### First Time Login

1. Click **"Login with Discord"** on the login page
2. A popup window opens with Discord's authorization page
3. Click **"Authorize"** to grant access
4. The popup closes automatically and you're logged in!

### What Discord Access Provides

- Your Discord username and avatar for your profile
- Verified email (from Discord) for account recovery
- Secure session that lasts 30 days

### Logging Out

Click **"Logout"** in the navigation bar. Your session will be cleared.

---

## Joining a Tribe

### Finding Tribes

1. Visit the **Tenants** page to see public tribes
2. Click on a tribe to view its public info
3. If the tribe allows new members, you'll see a join option

### Join Request Process

Most tribes require approval before you can join:

1. Navigate to the tribe's page
2. Click **"Request to Join"** (if available)
3. Fill out any required information:
   - **Character Name** - Your in-game or display name
   - **Wallet Address** - (Optional) Your Eve Frontier wallet
   - **Note** - Why you want to join
4. Submit your request
5. Wait for a tribe admin to approve

### After Approval

Once approved, you'll be able to:
- Access the tribe's member list
- View your rank and permissions
- Access any modules the tribe has installed

## Your Profile

Visit **"Profile"** or `/me` to see your account info:

- Display name (from Discord)
- Email (verified via Discord)
- Tribes you're a member of
- Your role in each tribe

---

## Tribe Features

Each tribe may have different modules installed. Common features include:

### Member List
View all tribe members, their ranks, and join dates.

### Ranks
Tribes can define custom ranks with different permissions:
- **Owner** - Full control over the tribe
- **Admin** - Can manage members and settings
- **Member** - Standard access
- Custom ranks defined by the tribe

### Access Lists
Some tribes use access lists for:
- Gate access in Eve Frontier
- Permission groups
- Role-based features

## Troubleshooting

### Discord Login Popup Doesn't Close

If the login popup doesn't close automatically:
1. Check if you're logged in on the main page
2. If yes, close the popup manually
3. If no, try the login again

### Session Expired

If you're suddenly logged out:
1. Your session may have expired (30 days)
2. Simply log in again with Discord

### Can't Access Tribe Features

If you see "Not authorized" or "Not a member":
1. Make sure you're logged in
2. Verify your join request was approved
3. Contact the tribe owner if issues persist
